Modus Ponens
P1 : Jika Toni mandi maka saya pergi kuliah
P2 : Toni mandi
K : Saya pergi kuliah

Modus Tollens
P1 : Jika Toni mandi maka saya pergi kuliah
P2 : Saya tidak pergi kuliah
K : Toni tidak mandi

Equivalence Elimination
P1 : Toni mandi jika dan hanya jika Ia pergi ke pasar
P2 : Jika Toni mandi maka ia pergi ke pasar
K : Jika Ia pergi ke pasar maka Toni mandi

Silogisme Disjungtif
P1 : Budi membeli es krim atau Bapak ke kantor
P2 : Budi tidak membeli es krim
K : Bapak ke kantor

Silogisme Hipotesis
P1 : Jika Toni mandi maka saya pergi kuliah
P2 : Jika saya pergi kuliah maka Jono tidur
K : Jika Toni mandi maka Jono tidur
